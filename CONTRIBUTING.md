# Guía para Contribuidores

## Open Research Collection — Juan Moisés de la Serna Tuya: Neurociencia, Economía Conductual, Salud Pública y Demografía en América Latina (2024-2025)

Agradecemos contribuciones a este proyecto de investigación abierta.

### Cómo contribuir

1. Abre un Issue describiendo la mejora o corrección propuesta
2. Haz un fork del repositorio
3. Crea una rama con un nombre descriptivo
4. Realiza los cambios y añade tests si aplica
5. Envía un Pull Request con descripción detallada

### Estándares

- Seguir los principios FAIR (Findable, Accessible, Interoperable, Reusable)
- Documentar todos los cambios en el CHANGELOG
- Mantener la licencia CC BY 4.0

### Contacto

Para consultas científicas: contactar al autor a través de Zenodo (https://doi.org/10.5281/zenodo.19151868)
