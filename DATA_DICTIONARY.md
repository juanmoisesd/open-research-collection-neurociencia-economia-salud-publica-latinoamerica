# Diccionario de Datos / Data Dictionary

## Open Research Collection — Juan Moisés de la Serna Tuya: Neurociencia, Economía Conductual, Salud Pública y Demografía en América Latina (2024-2025)

**DOI:** https://doi.org/10.5281/zenodo.19151868

| Campo | Tipo | Descripción |
|-------|------|-------------|
| titulo | string | Título del dataset o estudio |
| autor | string | Nombre del autor principal |
| año | integer | Año de publicación |
| doi | string | Identificador de objeto digital |
| area | string | Área temática (neurociencia, economía conductual, etc.) |
| pais | string | País o región de estudio |
| n_registros | integer | Número de registros en el dataset |
| fuente | string | Fuente original de los datos |
| licencia | string | Tipo de licencia (CC BY 4.0) |

## Notas

- Todos los datos son de acceso abierto bajo CC BY 4.0
- Los archivos CSV utilizan codificación UTF-8
- Los valores nulos se representan como cadenas vacías
